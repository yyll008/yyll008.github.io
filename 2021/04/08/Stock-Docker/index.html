Stock Docker

发表于 2021-04-08

https://github.com/pythonstock/stock

mkdir -p /data/mariadb/data
docker pull pythonstock/pythonstock:latest
docker pull mariadb:latest

docker run –name mariadb -v /data/mariadb/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=mariadb -p 3306:3306 -d mariadb:latest

docker run -itd –link=mariadb –name stock -v /data/notebooks:/data/notebooks -p 8888:8888 -p 9999:9999 pythonstock/pythonstock:latest

进入镜像：

docker exec -it stock bash
sh /data/stock/jobs/cron.daily/run_daily
